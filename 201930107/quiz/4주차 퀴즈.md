>1. 변수명 규칙
>> 1. 변수명 앞에 $를 붙인다.
>>> $value;
>> 2. 숫자로 시작할 수 없다.
>>> $123; X
>> 3. 글자로 시작해야 하지만 예외로 '_'를 시작기호로 허용함.
>>> $_value;
>> 4. 대소문자 구별함.
>>> $VALUE; $value;
>> 5. 한글 사용 가능함.
>>> $변수;

>2. ```$arr = [1, 2, 3, 4, 5, 6];``` 주어진 배열의 원소의 개수를 출력하시오
>> ```echo count($arr);```

>3. ```$value = 123;``` 주어진 변수를 가변변수를 이용하여 출력하시오
>> ```echo ${"value"};```
>>> 중괄호 사이에 문자열과 같은 이름을 가지는 변수 선택, 그 변수의 값 출력
