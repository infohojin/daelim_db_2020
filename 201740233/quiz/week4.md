## 1. 다음 보기중 올지 않은 문자열은?
```
1) $hello = "hello 'world'...";
2) $hello = 'hello "world"...';
3) $hello = "hello wolrd'
4) $hello = "hello \"world\"... ";

답: 3번 ("으로 시작한 문자열은 "로만 끝나야 하기 때문에)
```

## 2. 다음 보기중 올지 않은 변수명은?
```
1) $hello
2) $_hello
3) $hello123    
4) $123hello
5) $안녕

답: 4번 (변수명은 숫자로 시작할 수 없다)
```
