
1. 서버에 값을 전송하는 방식이 아닌 것은?
① SEND 방식  ② GET 방식  ③ POST 방식
정답: 1. SEND 방식
해설: GET방식과 POST방식은 서버에 값을 전송한다.

2. 데이터 전송을 기반으로한 요청 메서드이고, URL에 붙여서 보내지 않고
   Body 안에 데이터를 넣어서 보내는 방식은?
정답: POST 방식
해설: POST 방식은 URL에 표시되지 않고, Body 안에 값을 넣어 보낸다.

3. 웹 프로그램으로 입력 데이터를 전달하는 방식으로,
  아래와 같이 "?"(물음표) 뒤에 표현되는 부분은?
  -- http://localhost:8000/insta.php ? 변수명 = 값 & 변수명 = 값
정답: 쿼리스트링
해설: 위 형식에서 URL주소와 쿼리스트링은 "?" 로 구분되고,
      여러 개의 변수와 값을 전달할 경우 "&"로 구분한다.
