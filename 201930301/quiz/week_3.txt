1. PHP 주석이 아닌 것은?
① // 주석1 ② /* 주석2 */ ③ <!-- 주석3 --> ④ # 주석4
정답 : 3.<!-- 주석3 -->
해설 : <!-- -->은 HTML 주석이다.

2. 상수를 선언하는 const는 반복문 또는 if문 안에서 사용이 가능하다.
정답 : X
해설 : 함수나 반복문에서 중복된 상수명을 선언할 수 있기 때문에
       사용할 수 없다.

3. 상수의 종류에는 정수형, 실수형, 문자, 문자열이 있다
정답 : O
해설 : 상수의 종류에는 소수점 이하 부분이 없는 정수, 소수점이나 지수가 있는 실수,
       하나의 단일 문자, 두 개 이상의 문자가 모인 문자열이 있습니다.
